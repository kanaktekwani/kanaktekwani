# Kanak Tekwani

Software engineer focused on backend systems, cloud infrastructure, AI-enabled
applications, and automation.

I recently completed my Master's in Computer Science at Arizona State
University and enjoy building practical systems that connect reliable APIs,
cloud-native workflows, and AI features into usable products.

## Focus Areas

- Backend systems and API design
- Cloud infrastructure and DevOps workflows
- AI-enabled applications and automation
- Realtime systems and integration-heavy products
- Mobile and sensor-driven prototypes

## Featured Projects

### Voice Authenticator

Realtime Zoom App prototype for live speaker-authenticity status updates using a
Node/Express Zoom App shell, Python status service, MongoDB device registration,
and Server-Sent Events.

Repository: https://github.com/kanaktekwani/Voice-Authenticator

### OpenMonitor

Visual AI hackathon project for restricted-area monitoring. The system uses
computer vision to detect unauthorized access and supports realtime alerting,
event logging, and a web monitoring interface.

Repository: https://github.com/kanaktekwani/OpenMonitor  
Upstream: https://github.com/vibhu-dixit/OpenMonitor

### 3D Portfolio

Resume-facing portfolio built with React, TypeScript, Vite, Three.js, React
Three Fiber, and GSAP.

Live: https://kanaktekwani.github.io/portfolio/  
Repository: https://github.com/kanaktekwani/portfolio

### Health Monitoring App

Android prototype that estimates heart rate and respiratory rate from phone
sensors, logs symptoms, and stores measurements locally with Room.

Repository: https://github.com/kanaktekwani/HealthMonitoringApp

## Tech Stack
